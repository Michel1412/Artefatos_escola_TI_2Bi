Minha equipe de software : Classe Principal → Relatório

Created by Andre Felipe, last modified on jun. 01, 2026

Classe principal responsável por representar os relatórios gerados no sistema LogViewer. Ela consolida logs tratados, erros classificados, filtros aplicados e documentos exportados, permitindo que o usuário visualize informações resumidas sobre ocorrências, criticidade, período analisado e status dos erros.
