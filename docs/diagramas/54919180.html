Minha equipe de software : Classe: LLMService

Created by Michel Bocchi Junior on mai. 25, 2026

Descrição

Representa a interface responsável pela integração com modelos de IA generativa (LLM) utilizados pelo sistema LogViewer. Atua como ponto central de comunicação entre o Backend API e o provedor de IA, sendo responsável por montar o contexto empresarial do usuário, enviar logs para análise e retornar sugestões passivas de solução. A classe abstrai detalhes de integração externa e padroniza o fluxo de análise inteligente de erros conforme o UC-13.
