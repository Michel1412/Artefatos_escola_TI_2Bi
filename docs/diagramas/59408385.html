Minha equipe de software : Classe de Caso de Uso → Exportar Relatório

Created by Andre Felipe, last modified on jun. 01, 2026

Classe responsável por realizar a exportação de relatórios no sistema LogViewer. Ela valida o formato escolhido pelo usuário, gera o arquivo correspondente e disponibiliza o documento para download, permitindo que relatórios sejam exportados em formatos como PDF, CSV ou XLSX.
