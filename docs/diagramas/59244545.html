Minha equipe de software : Classe de Caso de Uso → Gerar Relatório

Created by Andre Felipe, last modified on jun. 01, 2026

Classe responsável por executar o processo de geração de relatórios no sistema LogViewer. Ela utiliza os filtros definidos pelo usuário, busca logs tratados e erros classificados, monta o relatório consolidado e salva o resultado para posterior visualização ou exportação.

Atributos

Atributo

Tipo

Visibilidade

Descrição

usuarioId

String

privado

Identifica o usuário que solicitou a geração do relatório

filtroAplicado

FiltroRelatorio

privado

Armazena os critérios utilizados para buscar os dados do relatório

logsSelecionados

List<LogTratado>

privado

Lista de logs tratados encontrados com base nos filtros aplicados

errosSelecionados

List<Erro>

privado

Lista de erros classificados incluídos no relatório

statusGeracao

Enum

privado

Status do processo de geração, como pendente, em processamento ou concluído

Método

Retorno
