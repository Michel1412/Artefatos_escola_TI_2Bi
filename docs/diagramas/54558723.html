Minha equipe de software : Classe Principal -> Gerenciador de Assinatura

Created by Vinicius Ferrarini, last modified on mai. 25, 2026

Descrição: Classe responsável por orquestrar o ciclo de vida financeiro e de acesso dos clientes (vinculados a um CNPJ). Atua no Core Domain da arquitetura hexagonal e se comunica com Adapters externos através de Ports de saída (Interfaces de Gateway de Pagamento). Implementa rigorosamente a RN-01, calculando o valor da assinatura dinamicamente com base na quantidade de sistemas cadastrados. Também gerencia os bloqueios por inadimplência, processa integrações via Webhooks de confirmação (RF-28) e valida as permissões administrativas necessárias para alterações financeiras (RN-13).
