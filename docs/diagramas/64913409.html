Diagrama de Classe Geral — Sistema LogViewer

Objetivo:
Documentar a visão geral das principais classes do sistema LogViewer, reunindo os módulos de autenticação, processamento de logs, inteligência artificial, notificações, relatórios, assinatura e pagamento.

Descrição:
Este diagrama apresenta uma visão unificada da estrutura de classes do sistema LogViewer. Ele foi criado a partir dos diagramas individuais desenvolvidos pelos integrantes da equipe e consolidado para representar os principais componentes do sistema de forma organizada.

Módulos representados:

  • Usuário e Autenticação

  • Logs e Processamento

  • IA e Notificações

  • Relatórios e Exportação

  • Assinatura e Pagamento

Ferramenta utilizada:
PlantUML
