title Fluxo de Ingestão - Sistema de Gestão de Logs

actor "Sistema Cliente (SDK)" as SDK
participant "Sistema de gestão de Logs" as API
participant "Clerk (Auth Provider)" as Clerk
participant "Backend LogViewer" as Backend

SDK->API: Enviar Log + Token
activate API

API->API: Mascarar Dados Sensíveis

API->Clerk: Validação de Sessão/User
activate Clerk
Clerk-->API: Retorno da Validação (Válido/Inválido)
deactivate Clerk

opt Token Válido
    API->Backend: Processamento Interno (Persistir Log)
    activate Backend
    Backend-->API: Confirmação de Persistência
    deactivate Backend
    
    API-->SDK: Sucesso (200 OK)
else Token Inválido
    API-->SDK: Erro de Autenticação (401 Unauthorized)
end

deactivate API
