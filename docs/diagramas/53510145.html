Minha equipe de software : Classe Principal -> Usuário

Created by pedroalvaro193, last modified on mai. 25, 2026

TR-75 - Criar o diagrama de classe - 3 Concluído

Descrição: Representa a entidade central de identidade do sistema LogViewer. Um Usuário é sempre vinculado a um CNPJ (pessoa jurídica). É responsável por carregar as configurações de acesso, preferências de notificação, nível de permissão e vínculo com o plano de assinatura ativo. Serve como agregado raiz no domínio de identidade, sendo referenciado por todas as demais entidades que exigem isolamento de dados.
