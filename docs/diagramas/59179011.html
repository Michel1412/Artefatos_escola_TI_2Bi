Minha equipe de software : Classe de Apoio → Filtro Relatório

Created by Andre Felipe, last modified on jun. 01, 2026

Classe de apoio responsável por armazenar os critérios utilizados na busca de logs e erros para geração de relatórios no sistema LogViewer. Ela permite filtrar os dados por período, sistema, tipo de erro, criticidade, status e texto de pesquisa, garantindo que o relatório seja gerado apenas com as informações relevantes para o usuário.

Enum

Define o nível de criticidade dos erros filtrados

statusErro

Enum

privado

Define se serão buscados erros solucionados, não solucionados ou ambos

textoPesquisa

String

privado

Texto utilizado para pesquisa livre nos logs ou mensagens de erro

Método

Retorno
