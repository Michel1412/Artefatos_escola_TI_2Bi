Minha equipe de software : Classe de Apoio → Documento Relatório

Created by Andre Felipe, last modified on jun. 01, 2026

Classe de apoio responsável por representar o arquivo gerado a partir da exportação de um relatório no sistema LogViewer. Ela armazena informações como nome do arquivo, formato, caminho de armazenamento, URL de download e data de criação, permitindo que o usuário acesse posteriormente o documento exportado.
