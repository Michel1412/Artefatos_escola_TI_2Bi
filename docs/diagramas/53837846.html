Minha equipe de software : Classe de Caso de Uso -> Serviço de Filtragem do Log

Created by pedroalvaro193, last modified on mai. 25, 2026

TR-75 - Criar o diagrama de classe - 3 Concluído

Descrição: Classe responsável por expor as capacidades de consulta, filtragem avançada e pesquisa sobre o histórico de logs persistidos no MongoDB. Atua como Port de entrada para as operações de leitura no domínio de logs, sendo consumida pelo Dashboard Web, pela interface Mobile e pela funcionalidade de relatórios. Viabiliza os filtros multidimensionais exigidos pelo RF-14 (por sistema, tipo de erro, status, criticidade, período, marcador), a busca em todo o histórico armazenado (RF-21), a visualização do contador de logs repetidos (RF-22) e o filtro mestre para visão por sistema específico ou multi-sistemas (RN-10). Todos os resultados filtrados podem servir como base para exportação em PDF e Excel (RF-24/RN-11).
