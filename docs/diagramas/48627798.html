title UC-08: 1. Carregamento Inicial do Histórico

actor "Usuário" as User
participant "Frontend\n(Web/Mobile)" as UI
participant "Backend API" as API
database "MongoDB" as DB

User -> UI: Acessa a tela de "Histórico de Erros"
activate UI

UI -> API: Solicita histórico (Paginado)
activate API

API -> API: Valida Sessão e Extrai CNPJ (Tenant ID)

API -> DB: Query de logs do CNPJ (Regra de Isolamento: RN-01)
activate DB

alt Cenário Principal: Erros Encontrados
    DB --> API: Retorna documentos paginados
    API -> API: Formata dados (Timestamp, Tipo, Status)
    API --> UI: JSON com lista de erros
    UI --> User: Renderiza a tabela/lista na tela
    
else Alternativa A1: Nenhum erro encontrado
    DB --> API: Retorna array vazio []
    API --> UI: 200 OK (Sem dados)
    UI --> User: Exibe interface amigável de "Estado Vazio"
end
deactivate DB
deactivate API
deactivate UI
@enduml

title UC-08: 2. Busca e Filtros Avançados

actor "Usuário" as User
participant "Frontend\n(Web/Mobile)" as UI
participant "Backend API" as API
database "MongoDB" as DB

User -> UI: Aplica filtro (Ex: Multi-sistemas, Status ou Pesquisa)
activate UI

UI -> API: Solicita histórico com parâmetros (search, status, systemId)
activate API

API -> API: Garante isolamento (CNPJ) e monta Query Otimizada

API -> DB: Executa busca utilizando índices do MongoDB
activate DB
DB --> API: Retorna lista filtrada
deactivate DB

API --> UI: JSON atualizado
UI --> User: Atualiza a interface com resultados refinados
deactivate API
deactivate UI
@enduml
