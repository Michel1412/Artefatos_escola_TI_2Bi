Os Fluxos de Exceção: Mostra o que acontece quando dá errado (Rejeição ou Timeout).

Diagrama 1: O Caminho Feliz (Aprovação com Sucesso)

@startuml

title 1. Caminho Feliz: Login Web com Aprovação Mobile (Sucesso)

actor "Usuário" as User
participant "Aplicação Web" as Web
participant "Backend API" as API
participant "Clerk Auth" as Clerk
participant "App Mobile" as Mobile

== Autenticação Primária (Clerk) ==
User -> Web: Informa login e senha
activate Web
Web -> API: Iniciar login
activate API
API -> Clerk: Encaminhar validação
activate Clerk
Clerk --> User: Solicita código 2FA (Email/SMS)
User -> Clerk: Informa código
Clerk --> API: Autenticação válida
deactivate Clerk

== Estado de Espera e Aprovação ==
API -> API: Cria solicitação com cronômetro
API -> Mobile: Notificação Push: "Tentativa de login"
activate Mobile
API --> Web: Status: "Aguardando Mobile..."

User -> Mobile: Aprova acesso via Biometria (Face ID/Touch ID)
Mobile -> API: Retorna status "Aprovado"
deactivate Mobile

API -> API: Finaliza autenticação
API --> Web: Dispara liberação do acesso
Web --> User: Redireciona para o Dashboard
deactivate API
deactivate Web
@enduml

title 2. Exceções: Rejeição Biométrica e Timeout

actor "Usuário" as User
participant "Aplicação Web" as Web
participant "Backend API" as API
participant "App Mobile" as Mobile

note over Web, API: O sistema já passou pelo Clerk e a Web está aguardando aprovação.

alt Exceção A: Usuário Rejeita o Acesso (RN-16)
    User -> Mobile: Clica em "Não sou eu / Rejeitar"
    activate Mobile
    Mobile -> API: Retorna status "Rejeitado"
    deactivate Mobile
    activate API
    
    API -> API: Cancela login e Registra Evento de Segurança
    API --> Web: Notifica: Acesso Negado
    activate Web
    Web --> User: Exibe alerta de bloqueio de segurança
    deactivate Web
    deactivate API

else Exceção B: Tempo Expirado / Timeout (RN-17)
    activate API
    API -> API: Cronômetro atinge o limite (Ex: 3 minutos)
    API -> API: Invalida a solicitação de login pendente
    
    API --> Web: Notifica: Tempo Expirado
    activate Web
    Web --> User: Exibe erro e solicita novo login do zero
    deactivate Web
    deactivate API
end
@enduml
