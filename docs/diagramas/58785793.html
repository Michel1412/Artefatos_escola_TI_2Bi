Minha equipe de software : Classe de Caso de Uso → Capturador de Erro via Interface

Created by Andre Felipe, last modified on jun. 01, 2026

Classe responsável por capturar erros informados manualmente pela interface do sistema LogViewer. Ela valida os dados preenchidos pelo usuário, associa o erro ao sistema selecionado e envia as informações para o fluxo de tratamento de erro. Essa classe permite que erros também sejam registrados pela interface, não apenas via SDK.
