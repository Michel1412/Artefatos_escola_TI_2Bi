' ===========================================================
'  RELACIONAMENTOS - AUTENTICAÇÃO
' ===========================================================
ClerkIdentity "1" --> "1" AuthenticationManager : fornece identidade\nautenticada ao manager
AuthenticationManager "1" --> "1" NotifyAuth2Factor : orquestra fluxo\nde 2FA quando ativado
NotifyAuth2Factor --> Status2FA : possui status
AuthenticationManager "1" --> "1" Usuario : resolve identidade\npara o domínio

' ===========================================================
'  RELACIONAMENTOS - USUÁRIO E SISTEMA
' ===========================================================
Usuario "1" --> "N" Sistema : cadastra e gerencia
Usuario "1" --> "1" ContextoEmpresa : configura contexto
Sistema "1" --> "1" ContextoEmpresa : aplica contexto\nno processamento

' ===========================================================
'  RELACIONAMENTOS - CAPTURA E INGESTÃO
' ===========================================================
CapturadorDeErroSDK ..> LogController : envia log via HTTP
CapturadorDeErroSDK ..> Log : cria instância
CapturadorErroInterface "1" --> "1" Sistema : vincula ao sistema
CapturadorErroInterface "1" --> "1" Erro : cria erro
CapturadorErroInterface ..> LogController : encaminha para ingestão
LogController ..> ProcessamentoDeLogs : delega processamento
LogController ..> Log : gerencia ciclo de vida
LogController ..> ServicoSugestao : solicita sugestão ao aceitar

' ===========================================================
'  RELACIONAMENTOS - PROCESSAMENTO E INTELIGÊNCIA
' ===========================================================
ProcessamentoDeLogs ..> Log : processa e enriquece
ProcessamentoDeLogs ..> ParserStrategy : usa estratégia de parse
ProcessamentoDeLogs ..> ServicoNotificacao : dispara notificação\napós processar
ProcessamentoDeLogs "1" --> "1" ServicoPersistenciaLog : persiste log\ntratado
ServicoTratamentoLog "1" --> "1" ContextoEmpresa : aplica contexto
ServicoTratamentoLog ..> LogTratado : produz log tratado
ServicoTratamentoLog "1" --> "1" ServicoNotificacao : dispara alerta\napós processar erro
SugestaoDeLogs ..> LLMService : delega análise ao LLM
SugestaoDeLogs ..> Log : analisa log para sugestão
ServicoSugestao "1" --> "1" ServicoNotificacao : notifica ao gerar\nsugestão
ServicoSugestao ..> SugestaoDeLogs : usa para gerar solução

' ===========================================================
'  RELACIONAMENTOS - USUÁRIO → SERVIÇOS CORE
' ===========================================================
Usuario "1" --> "N" ServicoTratamentoLog : aciona processamento\npelo contexto configurado
ServicoFiltragemLog "N" --> "1" Usuario : filtra logs por usuário
ServicoSugestao "N" --> "1" Usuario : apresenta sugestão\npara consentimento
Usuario "1" --> "N" CapturadorErroInterface : registra erro

' ===========================================================

ServicoPersistenciaLog ..> LogTratado : recebe e persiste

'  RELACIONAMENTOS - RELATÓRIOS E EXPORTAÇÃO
